How to Use

  • Enter X and Y coordinates as comma-separated values
  • Select a plot type (Line, Scatter, or Bar)
  • Click "Generate Plot" to create your graph
  • View and save your plot image

Features

  • Pure C++ implementation (no external libraries)
  • Server-side image generation in BMP format
  • Integer-aligned axis labels and grid
  • Three plot types: Line, Scatter, and Bar
